## Deployment on Replit

1. Fork this repository on Replit.
2. Create a **PostgreSQL** database in the Replit Resource pane.
    - Replit will automatically set the `DATABASE_URL` environment variable.
3. Create an **Object Storage** bucket in the Replit Resource pane.
    - No additional configuration is needed; the code uses the default bucket.
4. Run the application:
    - Use the `Run` button or `npm run dev` / `npm start`.

## Environment Variables

See `.env.example`.
- `DATABASE_URL`: Automatically managed by Replit for production.
- `NEXTAUTH_URL`: Your Replit application URL.
- `NEXTAUTH_SECRET`: Generate a secret key.
